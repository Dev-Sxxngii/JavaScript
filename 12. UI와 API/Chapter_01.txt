User Interface(UI): 유저 인터페이스
Application Programming Interface(API): 어플리케이션 프로그래밍 인터페이스

(UI)
예시: 웹 사이트
- 뒤로가기/앞으로가기 및 버튼 등 사용자가 시스템을 사용하기 위한 요소(인터페이스)

(API)
예시: javascript:alert("Hello world!");

alert(경고창)
- 닫기 버튼 | 경고 이미지 | 경고음 | 기능 잠금(확인 전)...

- 위와 같이 웹 개발자는 alert(); 한 줄 만으로도 다양한 기능을 포함한 경고창을 띄울 수 있다.
  이는 웹 개발자가 코드를 통해 시스템을 제어할 수 있게 브라우저에서 제공하는 API이다.

- 웹 개발자는 UI를 통해 사용자가 시스템을 사용할 수 있게 하며, 브라우저가 제공하는 API를 통해 시스템을 제어한다.

- 브라우저 개발자는 UI를 통해 개발자가 시스템을 제어할 수 있게 하며, 운영체제가 제공하는 API를 통해 시스템을 구축한다.

- 운영체제 > 브라우저 > 개발자 > 사용자
